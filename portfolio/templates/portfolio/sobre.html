Tecnologias Utilizadas

Backend: Django (Python)
Frontend: HTML5, CSS3 (com templates do Django)
Base de Dados: SQLite (desenvolvimento), MySQL (produção)
Serviços: PythonAnywhere (deployment)
Ferramentas: Git, GitHub, VS Code

Arquitetura da Aplicação

A aplicação segue o padrão arquitetural MVT (Model-View-Template):

Model: Define a estrutura dos dados (ex: Projeto, Tecnologia, Disciplina)
View: Processa requisições e seleciona os dados a exibir
Template: Define como os dados são apresentados ao utilizador

Modelação e Diagrama DER

A modelação da aplicação representa entidades como Projeto, Disciplina, Tecnologia e Imagem do Projeto. A relação entre os modelos é a seguinte:

Projeto tem uma relação ManyToMany com Tecnologia
Projeto tem uma relação ForeignKey com Disciplina
ImagemProjeto tem uma relação ForeignKey com Projeto
DetalhesExtras tem uma relação OneToOne com Projeto
